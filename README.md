# AI Retail Analytics Copilot

AI Retail Analytics Copilot is an AI-powered business intelligence application that allows users to explore retail and e-commerce data using plain English.

The project uses **Snowflake, dbt, Snowflake Cortex Analyst, Python, and Streamlit** to transform raw business data into trusted metrics, interactive dashboards, and conversational insights.

Users will be able to ask questions such as:

* What was net revenue last month?
* Which products generated the highest profit?
* Why did gross margin decrease?
* Which products are at risk of running out of stock?

The system will interpret the question, generate a governed SQL query, run it in Snowflake, and return:

* a natural-language answer,
* a result table,
* a chart,
* the applied filters,
* and the generated SQL.

The final application will include dashboards for sales, profitability, customers, marketing, products, returns, and inventory, along with an **Ask Your Data** chat interface.

This project benefits users by reducing the need to write SQL, making business data easier to access, and helping teams investigate trends, risks, and opportunities more quickly.

It also demonstrates a practical combination of data engineering, analytics engineering, semantic modeling, text-to-SQL, visualization, and AI-powered business intelligence.

The project will be completed as a focused three-week portfolio implementation using a realistic retail dataset and a governed analytics architecture.
